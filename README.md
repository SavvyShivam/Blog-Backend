### Key Features:
- User authentication: Only authorized users can access the Blog application.
- Blog Post:
  - Create Post
  - View Post.
  - Edit and update post.
  - Add new comments.
- Tested APIs using **Postman**.

## Tech Stack

- **Backend Framework**: Spring Boot
- **Database**:  MySQL
- **API Testing**: Postman
